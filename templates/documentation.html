Chat API Docs

Base url : https://chatlog-webtech-fa086797d6f9.herokuapp.com/

Endpoints

/get_users - GET

Returns JSON list of all unique users in the message database.

/get_messages - GET

Returns JSON of all messages in the database.

/new_message - POST

Creates a new message. Expects form data for 'message', 'from_user', and 'to_user'.

/update_message - PATCH

Updates a message by ID. Expects form data for 'id' and 'edited_message_text'.

/delete_message - DELETE

Deletes a message by ID. Expects form data for 'id'.

Example Requests

/get_users - GET

/new_message - POST

Form data:
message: Hello world!
from_user: alice
to_user: bob

/update_message - PATCH

Form data:
id: 5
edited_message_text: Hello there!
